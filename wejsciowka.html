Rozwiązanie wejściówki

Wczytanie danych z pliku csv, a zmiana nazw kolumn następuje później z wykorzystaniem funkcji names(<ramka.danych>) <- wektor nazw.

df <- read.csv2("data-raw/gus-nieruch-dane.csv")

names(df) <- c("kod", "nazwa", "trans", "pow", "rok", "wartosc", "jednostka", 
               "atrybut", "x")

Liczba wierszy

Jeżeli plik jest bardzo duży, można użyć argumentu n wskazując liczbę wierszy do wczytania.

df <- read.csv2("data-raw/gus-nieruch-dane.csv",
                col.names = c("kod", "nazwa", "trans", "pow", "rok", "wartosc", 
                              "jednostka", "atrybut", "x"), 
                nrows = 2)
head(df)
